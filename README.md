# Wood Cutting Optimizer - Streamlit v2

수정 사항:
- 재단 수량(qty) 직접 수정 가능
- 톱날폭 소수점 입력 가능
- 여유치 소수점 입력 가능
- 오류 메시지 표시 강화
- 엑셀 헤더 공백 처리 추가

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 배포
Streamlit Cloud에서 다음으로 설정하세요.

- Repository: 현재 GitHub 저장소
- Branch: main
- Main file path: app.py

## 주의
Python 버전은 3.10 또는 3.11 권장
